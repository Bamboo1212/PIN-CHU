      About Me 關於我

      我是品竹，一位兼具視覺美感與前端技術的網頁設計師，
      曾任網頁部門總監與設計主任。
      在網頁開發的生態圈中，設計師與工程師之間往往隔著一道「通靈牆」—— 設計師不懂程式限制，工程師討厭調整破版。而我，正是為了打破這道牆而生。

      憑藉多年帶領團隊、管控中大型商業專案的豐富資歷，我深諳如何兼具 Figma 視覺美感與前端切版技術。現在，我將這份總監級的專案視野帶入獨立接案。不論您是需要建立高質感門面的企業主，還是正在尋找靠譜前端戰友的工程團隊，我都能提供最高效、精準的網頁解決方案。

      What's Can I do? 我能為您解決什麼？

      • 對微型品牌與中小企業：用總監級視野，為您省下兩端溝通的無成本
      • 對工程師與開發團隊：總監級規範，交付最乾淨好套的靜態代碼
      01

      減半溝通成本

      設計與切版由我一人通包，溝通成本直接減半，且專案時程完全可控。

      02

      資產 100% 自主

      結案交付完整的「網頁原始碼檔案包」，不代購網域主機、不綁約、不託管，免除後續高額維護費糾紛。

      03

      SEO 優化

      嚴格遵循語意化 HTML 標籤結構與圖片效能優化，讓您的靜態形象站本身就具備極佳的 Google 搜尋潛力。

      04

      拒絕通靈盲猜

      設計稿嚴格遵循完備的 Design System 與 Auto Layout，完整標示 Hover/Active 等所有交互狀態。

      05

      模組化、不疊床架屋的 Code

      交付結構清晰、語意化且模組化的 HTML5 與 CSS，Class 命名符合邏輯與大型架構規範，極速拆解好套版。

      06

      完美 RWD 防破版

      嚴格控制卡片與短標籤的排版流動，確保在手機版到大螢幕等各式寬度下皆能優雅呈現，絕不折行破版。

      Graphic Design Services 精緻平面視覺服務

      除了網頁設計與切版，我的職涯也包含了深厚的實體與數位平面設計底子，讓我比一般純前端工程師更具備商業行銷美感。當您的網頁建置完成後，我也能為您提供一線品牌規格的平面延展服務，免去您再度尋找其他設計師的溝通內耗：

      • 數位行銷視覺：電商產品大圖、社群行銷 Banner、電子報 (EDM) 視覺規劃。
      • 實體品牌延展：高質感名片、精緻品牌酷卡、卡通授權商品及系列周邊視覺延展。

      Skills & Tools 技術與工具棧

      • 專案企劃與邏輯梳理：Miro / 專案時程控管 / 心智圖與樹狀圖 / 使用者流程圖 Flowchart
      • 網頁介面與規格規劃：Figma (Design System / Component / Auto Layout / 網頁響應式佈局規範)

      • 平面與行銷視覺：Adobe Photoshop & Illustrator (專業平面商務排版、實體印刷規範製版)
